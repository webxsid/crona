# Crona v1.4.0

Crona v1.4.0 is a prerelease that prepares the 1.4 line for broader validation with a denser Daily momentum summary, consistent check-in and away key handling, hard-limit timer state preservation, and configurable week-start settings.

## Highlights
- Daily summary momentum now shows compact `Signals` and `Momentums` sections so the check-in and focus ladders stay readable inside the Daily pane.
- The Daily and Wellbeing views now share the same `w` / `W` semantics, with `w` opening check-in and `W` toggling away mode where it applies.
- Timer state now preserves hard-limit pomodoro metadata end to end, including long-break timing and legacy prepared-state compatibility.
- Settings now support a configurable week start, and the calendar view uses that choice for its weekly layout.

## Fixed
- Long-break timer refreshes no longer lose their long-break duration metadata after a reload.
- Legacy prepared timer runtime files continue to load instead of being discarded as invalid.
- The settings view continues to route `Prompt Glyphs`, `Week Start`, and the recovery/telemetry sections correctly after the row layout shift.

## Notes
- This release is intended for beta validation of the 1.4 line.
