# Crona v1.2.0

Crona v1.2.0 adds a set of quality-of-life improvements across the daily workflow, habits, and reporting.

## Added
- Configurable date display formats for the TUI and Crona-generated reports.
- Prompt glyph modes for dialogs, with `emoji`, `unicode`, and `ascii` options.
- A Daily pin state for issues, alongside planned and overdue task handling.
- Custom habit streak definitions from Settings.
- A global habit history table that follows the current repo or stream context.

## Changed
- Daily summary bars use more of the available wide-screen space.
- Human-facing report dates follow the selected display format.
- Accountability numbers are rounded for readability.
- Global context and export actions are kept on the view line.
- Issue selection stays on the same issue as the list changes.

## Fixed
- Due-date calendar shortcuts now work reliably across terminals.
- Daily reporting no longer counts resolved or abandoned delayed work as active risk.
- Prompt glyph settings respond correctly in the Settings view.
- Habit history loads correctly from view navigation and respects the active context.
